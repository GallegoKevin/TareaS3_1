1. Encabezados

Es un elemento utilizado para definir títulos o subtítulos dentro de una página web. Los encabezados se representan mediante las etiquetas <h1>, <h2>, <h3>, <h4>, <h5> y <h6>, donde <h1> es el más importante y <h6> es el menos importante en términos de jerarquía.

Ejemplo:

codigo:

<h1>Encabezado Nivel 1</h1>

<h2>Encabezado Nivel 2</h2>

<h3>Encabezado Nivel 3</h3>

<h4>Encabezado Nivel 4</h4>

<h5>Encabezado Nivel 5</h5>

<h6>Encabezado Nivel 6</h6>

vista en el navegador:

Encabezado Nivel 1

Encabezado Nivel 2

Encabezado Nivel 3

Encabezado Nivel 4

Encabezado Nivel 5
Encabezado Nivel 6

2. Parrafo

es un elemento utilizado para estructurar y presentar texto en forma de bloques coherentes dentro de una página web. Se representa mediante la etiqueta <p> y se utiliza para agrupar y separar párrafos de texto relacionados.

Ejemplo:

codigo:

<p>"Este es el primer párrafo de texto en HTML"</p>

vista en el navegador:

"Este es el primer párrafo de texto en HTML"

3. Enlace

Es un elemento que permite a los usuarios navegar entre diferentes recursos web, como otras páginas HTML, imágenes, archivos de audio o video.

Se representa mediante la etiqueta <a> y se utiliza el atributo href para especificar la URL de destino del enlace.

Ejemplo:

codigo:

<a href="https://www.youtube.com" target="_blank">Visitar Youtube</a>

vista en el navegador

Visitar Youtube
