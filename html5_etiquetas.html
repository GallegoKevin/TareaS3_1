HEADER

se utiliza para agrupar contenido relacionado con la cabecera de una página web. Esto puede incluir elementos visuales como logotipos, títulos, menús de navegación y otros elementos que son consistentes en todas las páginas de un sitio web.

Lo representa <header></header>

NAV

se utiliza para definir una sección de navegación en una página web. Se utiliza para agrupar enlaces de navegación, como menús, barras de navegación, listas de enlaces y otras opciones de navegación.

Lo representa <article></article>

FOOTER

se utiliza para definir el pie de página de una página web. Se utiliza para incluir información como derechos de autor, enlaces de contacto, enlaces a políticas del sitio, créditos, información de contacto y otros elementos que se muestran al final de la página.

Lo representa <footer></footer>

SECTION

se utiliza para agrupar contenido temáticamente relacionado dentro de una página web. Se utiliza para dividir y organizar el contenido en secciones distintas y coherentes, lo que facilita la comprensión y la navegación del sitio para los usuarios.

Lo representa <section></section>

ARTICLE

se utiliza para marcar contenido independiente y autónomo dentro de una página web

Lo representa <article></article>

MAIN

se utiliza para marcar el contenido principal de una página web. Se utiliza para agrupar el contenido central y relevante de la página, como artículos, secciones, formularios u otro contenido importante que no esté relacionado con la cabecera, pie de página o navegación.

Lo representa <main></main>

ASIDE

se utiliza para marcar contenido relacionado pero independiente del contenido principal de una página web. Se utiliza para agrupar elementos como barras laterales, anuncios, notas explicativas, enlaces relacionados u otros contenidos que están relacionados con el contenido principal pero que no son parte esencial de él.

Lo representa <aside></aside>
