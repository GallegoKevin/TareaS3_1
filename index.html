1. Estructura básica DE HTML

<!DOCTYPE html>: Es una especificación del lenguaje y versión del documento HTML

<html lang="es"> </html>: Indica el comienzo y el final de un documento HTML

<head> </head>: Es un elemento que se utiliza para contener metadatos y enlaces a recursos externos que no son parte del contenido visible de la página web. Los elementos comunes que se encuentra dentro son: <title> </title>

<body> </body>: Corresponde al contenido de la pagina web el cual sera visible por los visitantes

Ejemplo de Estructura básica:
<!DOCTYPE html>
<html>
<head>
</head>
<body>
</body>
</html>

3. Tipos de Etiquetas

3.1 Etiquetas de Apertura y Cierre

Son etiquetas que marcan el inicio y final de un elemento

Ejemplo:

Codigo html:

<u>Este es un texto subrayado</u>

Vista en Navegador:

"Este es un texto subrayado"

Codigo html:

<i>Estos es una cursiva</i>

Vista en Navegador:

"Esto es una cursiva"

3.2 Etiqueta Self Closing

Etiquetas auto-cerradas (también conocidas como etiquetas vacías) son elementos que no requieren una etiqueta de cierre separada

Tales como:

<br>: Se utiliza para insertar un salto de línea o un quiebre de línea.

<img src="ruta/de/la/imagen.jpg" alt="Descripción de la imagen">: Se usa para mostrar imagenes en la pagina web

Ejemplo del codigo "br" :

Codigo html:

<p>Este es un texto con <br> un salto de línea</p>

Vista en Navegador:

Este es un texto con
un salto de línea.

Ejemplo del codigo "img" :

Codigo html:

<p>img src="https://miro.medium.com/v2/resize:fit:200/1*lXrmBn-3TzhkSgXatEU7iA.png" alt="self closing html">

Vista en Navegador:

3.3 Etiqueta bloque

Son etiquetas que ocupan todo el ancho disponible en la página, separando a los elementos que las contienen.

Tales como: <h1>; <table>; <ul>; <ol>

Ejemplo de vista en navegador:

<h4>:

Encabezado Nivel 4

<ul> & <ol>:

  • Elemento 1
  • Elemento 2
  1. Elemento 1
  2. Elemento 2

3.4 Etiqueta Inline

Son elementos que se alinean en la misma línea.

Tales como: <span>, <strong>, <em>, etc

Ejemplo de vista en navegador:

<span>:

Este es un texto con un texto en rojo dentro.

<strong>, <em>:

Texto en negrita y texto en cursiva.

4. Atributos

4.1 Obligatorios

Son aquellos que siempre deben tener un valor en una etiqueta HTML

<input>; El atributo "type" es obligatorio en la mayoría de los casos para especificar el tipo de campo de entrada.

Ejemplo:

codigo:

<input type="text" placeholder="Nombre">

Vista en Navegador:

<img>; El atributo "src" es obligatorio para especificar la ruta de la imagen que se mostrará.

Ejemplo:

<img src="ruta/de/la/imagen.jpg" alt="Descripción de la imagen">

4.2 Opcionales

Estos atributos modifican la funcionalidad por defecto de la etiqueta HTML

<a>; El atributo target; es opcional y sirve para controlar cómo se abre el enlace (en la misma ventana, en una nueva pestaña,etc.)

<a href="#" target="_blank">Abrir enlace en nueva pestaña<>
