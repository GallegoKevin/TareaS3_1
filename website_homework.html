Concepto de HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para la creación de páginas web. Permite estructurar y presentar contenido en la web utilizando etiquetas predefinidas.

Etiquetas HTML

HTML utiliza diversas etiquetas para definir la estructura y contenido de una página web. Algunas de las más comunes son:

  • <header>: Define el encabezado de la página.
  • <nav>: Define la barra de navegación.
  • <ul>: Define una lista desordenada.
  • <li>: Define un elemento de lista.
  • <a>: Define un enlace.
  • <img src="ruta_imagen.jpg" alt="Texto alternativo">: Inserta una imagen con texto alternativo.
  • <hr>: Inserta una línea horizontal.
  • <br>: Inserta un salto de línea.
  • <h1>: Define un encabezado de nivel 1.
  • <h2>: Define un encabezado de nivel 2.
  • <h3>: Define un encabezado de nivel 3.
  • <p>: Define un párrafo de texto.
  • <aside>: Define contenido relacionado pero secundario.
  • <footer>: Define el pie de página.
  • <strong>: Resalta un texto como importante.
  • <button type="button">: Define un botón interactivo.
  • <main>: Define el contenido principal.
  • <section>: Define una sección de contenido.
  • <article>: Define un artículo independiente.

Ejemplos de Uso

A continuación, se muestran ejemplos de cómo usar algunas etiquetas HTML:

Ejemplo de Botón

Puedes crear un botón interactivo utilizando la etiqueta <button>. Por ejemplo:

<button type="button">Click Aquí</button>

Ejemplo de Texto Importante

Para resaltar un texto como importante, utiliza la etiqueta <strong>. Por ejemplo:

<strong>Este es un texto importante.</strong>
